# SkillMirror 🎓

**SkillMirror** is an e-learning platform designed to help students learn, practice, and grow their skills in one place. It brings together courses, quizzes, video lectures, and progress tracking to make self-paced learning simple, engaging, and effective.

---

## ✨ Features

- 📚 **Courses** – Browse and enroll in structured courses across different skills/subjects
- 📝 **Quizzes & Assessments** – Test knowledge with interactive quizzes after each module
- 🎥 **Video Lectures** – Learn through recorded or streamed video content
- 📊 **Progress Tracking** – Track completion status, scores, and learning streaks
- 🏆 **Certificates** – Earn certificates on course completion *(if applicable)*
- 👤 **Student Dashboard** – A personalized space to manage enrolled courses and progress

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+) / Python (v3.x) — depending on your stack
- npm or yarn
- Database (MongoDB / PostgreSQL / etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/skillmirror.git
   cd skillmirror
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**

   Create a `.env` file in the root directory:
   ```
   DATABASE_URL=your_database_url
   JWT_SECRET=your_secret_key
   PORT=5000
   ```

4. **Run the application**
   ```bash
   npm start
   ```

5. Open your browser and visit `http://localhost:3000`
